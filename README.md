# Banking-System---OOP and CRUD
Simple banking project built in understanding OOP with deposit, withdrawal, balance update and transaction flow.
This project covers how a bank account work's backend 
-----
### Purpose
---To understand how Java OOP concepts are used in a particular scenario

Bank account is a good example cause it has both data and actions.

```text
Data   -account number, holder name, account type, balance
Actions   -deposit, withdraw, check balance

## Project Focus Areas

This project focuses on understanding:

* Object-Oriented Programming
* Business Logic Design
* Banking Transaction Flow
* CRUD Operations
* Validation Rules
* Software Design Thinking

The implementation intentionally starts with a simple design to build a strong foundation before introducing advanced concepts such as inheritance, polymorphism, abstraction, databases, APIs, and frontend integration.
